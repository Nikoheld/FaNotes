OneNote parser regression fixture
=================================

`Open Notebook.onetoc2` and `New Section 1.one` are the compact public test
notebook distributed with `onenote_parser` 1.1.1. They are used only by the
FaNotes import regression test and are covered by the upstream Mozilla Public
License 2.0 in `resources/onenote/LICENSE-onenote-parser.txt`.

Source: https://github.com/msiemens/onenote.rs/tree/v1.1.1/tests/samples
